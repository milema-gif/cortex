   - `let sessionAcked = false;`
   - `export function isAcked(): boolean { return sessionAcked; }`
   - `export function resetAck(): void { sessionAcked = false; }`

2. **reconcileRetry(db, observationId) -> ReconcileResult**:
   - Query sync_failures for the observation_id
   - If not found: return `{ success: false, message: "Observation {id} not found in sync_failures" }`
   - If found but status is 'pending': return `{ success: false, message: "Observation {id} is already pending retry" }`
   - If found and status is 'parked': UPDATE status to 'pending', reset attempt_count to 0
   - Return `{ success: true, message: "Observation {id} re-queued for retry" }` with current health
   - Recompute health after the change using computeHealth(db)

3. **reconcileDrop(db, observationId) -> ReconcileResult**:
   - Query sync_failures for the observation_id
   - If not found: return `{ success: false, message: "Observation {id} not found in sync_failures" }`
   - DELETE the row from sync_failures
   - Log: `log("info", "reconcile: dropped observation {id} from sync tracking")`
   - Return `{ success: true, message: "Observation {id} permanently dropped from sync tracking" }` with current health

4. **reconcileAck(db) -> ReconcileResult**:
   - Set `sessionAcked = true`
   - Log: `log("warn", "reconcile: blocked state acknowledged -- mutating operations resumed for this session")`
   - Return `{ success: true, message: "Blocked state acknowledged. Mutating operations will proceed for this session." }` with current health
